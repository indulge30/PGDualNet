# PGDualNet
深度学习方法已广泛应用于极化合成孔径雷达（PolSAR）土地覆盖分类。然而，其性能常受到信息损失以及由相似散射机制引起的视觉相似模式的制约。相比之下，直接利用丰富的极化分解特征则面临新的挑战，因为这些特征存在高度冗余，且难以有效识别最具判别性的特征子集。为解决这些问题，本文提出了一种新颖的物理引导双分支网络。该网络包含一个处理Pauli图像的空间分支，以及一个专门处理23维极化特征的极化分支。在极化分支中，我们设计了一种局部分组自适应特征选择机制，能够在端到端训练过程中动态选择判别性特征。关键之处在于，我们引入了多族多样性约束，依据特征背后的物理散射机制对极化特征进行分组。通过施加分组正则化、稀疏正则化以及物理相关正则化，确保所选特征具有物理一致性和互补性。在两个农业数据集（AIRSAR和RADARSAT-2）上的实验表明，所提方法在总体精度（OA）和平均交并比（mIoU）上显著优于现有最先进方法。此外，可视化分析证实，所选特征与不同作物类型的散射机制高度吻合，显著提升了分类性能和物理可解释性。
